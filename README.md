# Stock Alert - ストップ高狙いデイトレード支援アプリ

## 起動方法

```bash
cd stock-alert-app/backend
pip install -r requirements.txt
python -m uvicorn main:app --host 127.0.0.1 --port 8000 --reload
```

ブラウザで http://localhost:8000 にアクセス。

## 基本操作

### マジックワードの登録

1. 左パネルの「キーワード」欄にワードを入力
2. 「カテゴリ」欄にテーマ分類を入力（任意）
3. 「追加」ボタンまたはEnterキーで登録

### アラートの確認

- 30秒間隔でTDnet適時開示RSSを自動巡回
- キーワードがヒットすると右パネルに即時表示（緑フラッシュ）
- ヒットしたキーワードがタイトル内で緑色ハイライト表示
- ニュースタイトルをクリックすると元記事（PDF等）を新タブで開く

### 通知機能

- **音声アラート**: 新着時にビープ音（ヘッダーの🔔で ON/OFF 切替）
- **ブラウザ通知**: デスクトップ通知が表示される（初回アクセス時に許可を求められます）
- **WebSocket**: リアルタイム接続（ヘッダーの緑丸●で接続状態を確認）

### フィルタリング

- **カテゴリフィルタ**: マジックワード一覧をカテゴリ別に絞り込み
- **キーワードフィルタ**: アラートフィードを特定キーワードのみに絞り込み
- **全て既読**: 一括でアラートを既読にする

## 運用のコツ

1. **朝8:30〜9:00に起動** - 寄り前の適時開示ラッシュをキャッチ
2. **15:00〜15:30も注目** - 引け後の適時開示は翌日の材料
3. **キーワードは具体的に** - 「修正」より「上方修正」の方がノイズが少ない
4. **不要なアラートが多いキーワードは無効化** - 緑丸ボタンでON/OFF切替
5. **アラートが出たらすぐに関連銘柄を確認** - ストップ高の2番手・3番手を探す
6. **カテゴリを活用** - テーマ別に整理すると管理しやすい

## 技術情報

- バックエンド: Python FastAPI + SQLite (WAL mode)
- フロントエンド: HTML + Tailwind CSS + Vanilla JS
- 通信: WebSocket (リアルタイム) + REST API
- RSS監視: feedparser (30秒間隔)
- データソース: TDnet適時開示 (Yanoshin API)
